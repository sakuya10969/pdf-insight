# 仕様一覧

## SP-01: PDFアップロード・取り込み
- ドラッグ&ドロップまたはファイル選択でPDFアップロード
- バックエンドでPyMuPDF4LLMによるPDF解析
- 構造ベース + semantic補助でチャンク生成
- 埋め込み生成 → Qdrant登録
- PDFとJSONをローカルファイルシステムに保存
- ドキュメント一覧にステータス付きで表示

## SP-02: ドキュメント管理
- アップロード済みドキュメントの一覧表示（ステータス付き）
- ドキュメント削除（PDF、JSON、Qdrantベクトルを一括削除）
- メタデータ表示（チャンク数、アップロード日時）

## SP-03: 検索・チャット
- 自然言語クエリの入力
- クエリ埋め込み → ベクトル検索（top-k=3）
- マッチしたチャンクをソースカードとして表示
- LLMがコンテキストから回答を生成
- ソース参照付きで回答を表示

## SP-04: ルートレイアウト・ナビゲーション
- root.tsxでAppShellベースのレイアウト
- ヘッダーにアプリタイトル
- サイドバーナビゲーション（アップロード、検索、ドキュメント）
- レスポンシブ対応（モバイルでnavbar折りたたみ）

## SP-05: バックエンドインフラ
- FastAPIアプリのモジュラーなルーター登録
- Qdrant用docker-compose設定
- ヘルスチェックエンドポイント
- フロントエンド向けCORS設定
